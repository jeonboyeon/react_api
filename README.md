# Getting Started with Create React App

## 리액트 설치
npx create-react-app ./

## node-sass 설치
npm install node-sass

## router 설치 
<!-- 가상 주소를 만들어 준다. -->
npm install react-router-dom

## axios 설치
<!-- API를 편하게 쓰기위해 설치 -->
npm install axios

## gasp 설치
npm install gasp
